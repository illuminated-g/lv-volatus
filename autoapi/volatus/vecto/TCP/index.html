sendMsg(target: str | int, msgType: str, payload: bytes, sequence: int, task: str = '') → int¶

Enqueues a message to be sent and returns the approximate size of the message queue.

Messages that are sent while not connected are discarded.

Parameters:

  • target (str | int) – The ID or name of the target to send the message to.

  • msgType (str) – The message ID that identifies the data type and purpose of the message being sent.

  • payload (bytes) – A serialized message to be embedded within the TCP message that gets sent. Typically will be a serialized protobuf message.

  • sequence (int) – The count of payloads that have been sent from this application.

  • task (str, optional) – The specific task to send the message to in the target. If not specified will dispatch based on message ID, defaults to ‘’

Raises:

ValueError – Raised when an invalid target is specified.

Returns:

The approximate queue size of messges to be sent.

Return type:

int
