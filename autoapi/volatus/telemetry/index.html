class volatus.telemetry.Telemetry¶
createPublishGroupCfg(groupCfg: volatus.config.GroupConfig) → ChannelGroup¶
subscribeToGroupCfg(groupCfg: volatus.config.GroupConfig, timeout_s: float = None, bindAddress: str = '0.0.0.0') → tuple[ChannelGroup, bool]¶

_summary_

Parameters:

  • groupCfg (GroupConfig) – The configuration of the group to subscribe to. Must include publish configuration.

  • timeout_s (int, optional) – Wait up to this amount of time for data to arrive after subscribibg, defaults to None

Raises:

ValueError – The group config does not have a publish configuration.

Returns:

The group that was subscribed to and true if data has been received before the timeout.

Return type:

tuple[ChannelGroup, bool]
